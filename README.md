This web application was completed as a second-year second semester university project with 8 members for a client Hamilton De Silva and Sons, a company in Panadura, had trouble with inconsistent systems and manual processes, making it hard to run and make decisions. To fix this, we built an advanced ERP management system with key features like CRM, User Management, Project Management, Financial Management, Expense Tracking, Vehicle Management,Lone Managment and Inventory Management. This system integrates everything into one platform, making operations smoother and more efficient .my feature is vehicle management

MERN Stack

GitHub & Gihub desktop
Html/css/java script
